WELCOME TO REST-API SERVICE

SEVERAL URL ADDRESSES TO RECEIVE DATA BY GET METHODS: (Click on them)

http://localhost:8888/api/countries

http://localhost:8888/api/docs

http://localhost:8888/api/organization/{id}
Default id is equals 1 (Change id in http://localhost:8888/api/organization/{id} to your Number)

http://localhost:8888/api/user/{id}
Default id is equals 1 (Change id in http://localhost:8888/api/user/{id} to your Number)

http://localhost:8888/api/office/{id}
Default id is equals 1 (Change id in http://localhost:8888/api/office/{id} to your Number)
